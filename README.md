# Sistema de Biblioteca Universitaria - Examen Unidad 2
Materia: Programación / Estructuras de Datos

 Descripción del Proyecto
Este programa en C++ simula un sistema integral de biblioteca que incluye:
Gestión de Libros:** Registro, préstamo y devoluciones.
Lógica de Negocio:** Cálculo automático de multas y bloqueos temporales de usuario.
Módulo Dinámico:** Incluye un juego del Ahorcado basado en los títulos de la biblioteca.

 Temas Aplicados
 Programación Orientada a Objetos (POO).
Estructuras de control y validación de datos.
Manejo de lógica de bloqueos y condicionales.
